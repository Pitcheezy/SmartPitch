## 2. 병목 지점 분석

### 병목 1: MLP 정확도 57.5% (영향도: **최고**)

모든 하류 컴포넌트(MDP, DQN, PitchEnv)가 MLP의 `predict_proba()` 출력에 의존.
57.5% 정확도 = **42.5%의 잘못된 전이 확률** 위에서 정책을 학습.

| 하류 영향 | 설명 |
|-----------|------|
| MDP | 기대값 계산에 부정확한 확률 사용 -> 정책 왜곡 |
| DQN | 시뮬레이터 편향에 오버피팅 (Cease 사례) |
| PitchEnv | 잘못된 결과 샘플링 -> 비현실적 보상 |

**개선 여지 추정**:
- 데이터 3배 확장 (3시즌): val_acc +3~5pp -> 61~63%
- 추가 피처 (spin_rate, release_pos): val_acc +1~2pp
- Calibration 개선 (Temperature Scaling): 확률 품질 향상 (정확도 변화 없이 MDP/DQN 개선)

### 병목 2: 인플레이 타구 확률 하드코딩 (영향도: **높음**)

`pitch_env.py:415-428`과 `mdp_solver.py`에 70/15/10/5% 고정값 사용.
2023 MLB 실데이터 기준:
- 아웃: ~68% (근접)
- 1루타: ~16% (근접)
- 2루타: ~5% (현재 10%, **2배 과대**)
- 3루타: ~0.5% (현재 미포함)
- 홈런: ~3.5% (현재 5%, **1.4배 과대**)

**영향**: 홈런/2루타 과대 추정 -> RE24 보상 왜곡 -> DQN이 보수적 전략 과잉 학습

### 병목 3: RE24 2019 하드코딩 (영향도: **중간**)

2024-2025 시즌 투수에 2019 기대 실점 적용 중.
2024 MLB은 2019 대비 약 6% 낮은 득점 환경 (docs/re24_decision.md).
상대 비교에는 영향 없으나, 절대 보상값의 현실성 저하.

### 병목 4: 학습-추론 구종 분류 불일치 (영향도: **중간**)

| 단계 | 구종 분류 기준 |
|------|--------------|
| MLP 학습 (universal_model_trainer.py) | Statcast `pitch_type` (MLB 공식 라벨) |
| 추론 (main.py -> clustering.py) | 물리적 군집화 결과 (UMAP+KMeans) |

같은 투구가 학습 시 "Cutter"로, 추론 시 "Slider"로 분류될 수 있음.

### 병목 5: DQN 학습 부족 (영향도: **낮음**)

300K timesteps / 9,216 상태 = 상태당 평균 32.6회 방문.
탐색이 불충분하여 군집 1~3에서 MDP 대비 열위.
통계적 유의성 검증에도 1,000 에피소드 부족 (17,000+ 필요).

### 병목 6: PitchEnv 상태 초기화 비현실성 (영향도: **낮음**)

아웃(0/1/2), 주자(8조합), 타자군집(0~7) 모두 균등 분포로 초기화.
실제 야구에서는 0아웃이 가장 빈번하고, 주자 없음(000)이 ~55%.
DQN이 비현실적 상태 분포에서 학습 -> 현실 적용 시 편향 가능.

## 4. 사용하지 않는 Statcast 피처 (개선 후보)

현재 MLP가 사용하지 않지만 Statcast에서 제공하는 유망 피처:

| 피처 | 범위 | 기대 효과 | 우선순위 |
|------|------|----------|----------|
| `release_pos_x/z` | -3~+3 / 5~7 ft | 투수 팔 각도, 릴리스 포인트 변동 | **높음** |
| `spin_rate` | 1000~3500 RPM | 회전수 -> 무브먼트 예측 | **높음** |
| `spin_axis` | 0~360 deg | 회전축 -> 공 궤적 결정 | 중간 |
| `release_extension` | 1.0~3.5 ft | 체감 구속 영향 | 낮음 |
| `stand` (L/R) | 좌/우타 | 매치업 효과 (현재 batter_cluster에 암묵적 포함) | 낮음 |
| `p_throws` (L/R) | 좌/우투 | 매치업 효과 (현재 pitcher_cluster에 암묵적 포함) | 낮음 |

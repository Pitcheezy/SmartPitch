# RE24 매트릭스 변경 필요성 판단

## 1. 현재 상태

두 곳에 동일한 값이 정의되어 있습니다:

- `src/pitch_env.py` (PitchEnv.RE24_MATRIX, line 95)
- `src/mdp_solver.py` (MDPOptimizer.re24_matrix, line 81)

## 2. RE24의 역할

RE24는 SmartPitch에서 **보상 함수**로 사용됩니다:

```
reward = RE24(이전 상태) - RE24(현재 상태) - 득점
```

투수 시점에서 기대 실점을 줄이는 것이 목표이므로:
- 아웃을 잡으면 RE24 감소 → **양의 보상**
- 볼넷/안타를 허용하면 RE24 증가 → **음의 보상**
- 이닝 종료(3아웃) 시 RE24=0으로 수렴

RE24는 MDP 가치 반복(Value Iteration)과 DQN 학습 모두에서 동일한 보상 함수로 사용됩니다.

## 3. 2019 RE24 vs 2024 RE24 차이 분석

### 3.1 MLB 리그 전반 득점 환경 변화

| 시즌 | 경기당 평균 득점 | 주요 규칙 변경 |
|------|-----------------|--------------|
| 2019 | 4.83 | — |
| 2023 | 4.62 | 피치클락, 베이스 확대, 수비시프트 제한 |
| 2024 | 4.52 | 피치클락 2년차 |

2024 시즌은 2019 대비 약 **6.4% 낮은 득점 환경**입니다.
이는 RE24 값이 전반적으로 소폭 낮아졌음을 의미합니다.

MLB 득점 환경이 약 6% 낮아졌으므로, RE24 값도 대략 비례하여 낮아집니다:

|-------------|-----------|-----------------|------|
| 주자 없음 | 0.481 | ~0.450 | -0.031 |

## 4. 변경 시 예상 효과

### 4.1 평균 보상의 절대값

**약간 변할 수 있음.** 보상 = RE24(이전) - RE24(현재) - 득점이므로, RE24 스케일이 바뀌면 보상의 절대 크기가 변합니다. 다만 상대적 비율은 거의 동일하므로 변화 폭은 작습니다.

- 현재 Cole DQN: +0.436 → 2024 RE24로: 약 +0.410 예상 (추정)
- 절대값 차이: ~5-7%

### 4.2 표준편차

**거의 변하지 않음.** 표준편차는 야구의 본질적 변동성(운에 의한 결과)에 주로 기인합니다:

- 이닝 길이 변동 (3아웃 vs 10+타석)
- 인플레이 결과 확률론적 샘플링 (70% 아웃, 15% 1루타, 10% 2루타, 5% 홈런)
- 타자 군집 랜덤 배정

RE24 스케일이 6% 줄어도 이 분산 구조는 동일하므로, 표준편차도 ~6% 정도만 비례 감소합니다.

### 4.3 모델 간 상대 비교

**동일할 것.** RE24 매트릭스는 모든 에이전트(Random, MDP, DQN 등)에 동일하게 적용됩니다.
보상 함수의 스케일이 바뀌어도 상대 순위와 차이의 방향은 보존됩니다.

- DQN > MDP > Frequency 순서는 RE24 버전에 무관
- Cohen's d, t-test 결과도 실질적으로 동일

### 4.4 DQN 재학습 필요성

RE24 매트릭스를 변경하면 **보상 함수가 바뀌므로 DQN 재학습이 필요합니다.**
기존 모델은 2019 RE24 기준으로 최적화되었으므로, 2024 RE24에서는 약간 suboptimal합니다.
다만 차이가 ~6%이므로 정책 자체가 크게 달라지진 않습니다.

## 5. 변경 권장 여부

### 시간 여유가 있는 경우: **변경 권장**

- 정확성 향상: 2024+2025 시즌 데이터로 학습하면서 2024 RE24를 사용하는 것이 일관성 있음
- 발표 시 "2019 RE24를 사용하는 이유"에 대한 질문을 피할 수 있음
- 변경 작업량: `pitch_env.py`와 `mdp_solver.py` 두 파일의 딕셔너리 값만 교체 + 모델 재학습

### 시간이 부족한 경우: **그대로 사용**

- **모델 학습 자체에 영향 없음**: DQN은 MLP 시뮬레이터 위에서 학습하며, MLP는 투구 결과 전이 확률을 예측합니다. RE24는 MLP 학습에 사용되지 않습니다.
- **모델 간 비교에 영향 없음**: 모든 에이전트가 동일한 RE24를 사용하므로 공정한 비교입니다.
- **절대값 차이 미미**: 5-7% 수준이며 결론에 영향을 주지 않습니다.
- 발표 시 "2019 RE24를 기준으로 사용하며, RE24 버전은 상대 비교에 영향을 주지 않습니다"라고 명시하면 충분합니다.

## 6. 결론

| 항목 | 판단 |
|------|------|
| 모델 학습 품질 영향 | **없음** (MLP는 RE24 무관) |
| 에이전트 간 상대 비교 | **동일** (동일 보상 함수) |
| 절대 보상값 차이 | ~5-7% (미미) |
| 변경 시 재학습 필요 | **Yes** (DQN + MDP 모두) |
| **권장** | **현재 상태 유지** (발표 시간 고려) |

> **TL;DR**: RE24 2019→2024 변경은 "nice to have"이지 "must have"가 아닙니다.
> 모든 에이전트에 동일한 기준이 적용되므로 상대 비교에는 영향이 없습니다.
> 시간이 허락하면 변경하되, 발표 전에는 현재 상태를 유지하는 것이 안전합니다.
